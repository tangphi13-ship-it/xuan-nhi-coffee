1

Tạo tài khoản GitHub

Nếu bạn chưa có tài khoản GitHub, hãy đăng ký miễn phí:

🔑 Đăng ký GitHub

Hướng dẫn:

  • Nhập email, mật khẩu, tên người dùng (ví dụ: xuan-nhi)
  • Xác nhận email qua hộp thư đến
  • Quan trọng: Nhớ tên tài khoản (username) của bạn!
💡 Nếu đã có tài khoản, bỏ qua bước này và chuyển sang Bước 2
2

Tạo Repository (Kho chứa)

Bấm nút bên dưới để tạo kho chứa cho website:

📦 Tạo Repository Mới

Nhập thông tin như sau:

  • Repository name: xuan-nhi-coffee
  • Description (tùy chọn): Web quán cà phê Xuân Nhi
  • Public: ✅ Chọn Public (quan trọng!)
  • ✅ Tích chọn Add a README file
  • Bấm nút Create repository màu xanh
Tên repository: xuan-nhi-coffee
3

Upload file lên GitHub

Sau khi tạo repository xong, bạn sẽ thấy trang repository. Làm theo các bước sau:

Cách 1: Upload trực tiếp trên web (Dễ nhất)

  • Bấm nút Add file ➡ Upload files
  • Kéo thả các file sau từ folder Web vào:
    • index.html
    • style.css
    • script.js
    • Folder anh/ (kéo cả folder)
  • Kéo xuống dưới, bấm Commit changes

Cách 2: Upload từng file

  • Bấm Add file ➡ Create new file
  • Đặt tên file là index.html
  • Copy nội dung từ file index.html trên máy vào
  • Bấm Commit new file
  • Làm tương tự với style.css và script.js
📂 Mở GitHub của bạn
4

Bật GitHub Pages

Sau khi upload file xong, làm theo các bước sau:

  • Vào repository xuan-nhi-coffee của bạn
  • Bấm vào tab Settings (ở trên cùng)
  • Ở menu bên trái, bấm Pages
  • Ở mục Branch, chọn main (hoặc master)
  • Bên cạnh chọn / (root)
  • Bấm Save
  • Đợi 1-2 phút, trang sẽ được public!
⚙️ Mở Settings GitHub
⏳ Sau khi bật, đợi khoảng 2 phút là website hoạt động!
5

Kết quả - Link website của bạn

✅ Website của bạn sẽ có link:

https://[ten-tai-khoan].github.io/xuan-nhi-coffee

Ví dụ: Nếu tài khoản là xuan-nhi thì link là:
https://xuan-nhi.github.io/xuan-nhi-coffee

Cách tìm link sau khi bật:

  • Vào Settings ➡ Pages
  • Ở trên cùng sẽ hiện dòng chữ xanh:
    "Your site is published at https://xxx.github.io/xuan-nhi-coffee/"
  • Click vào link đó để mở website!
